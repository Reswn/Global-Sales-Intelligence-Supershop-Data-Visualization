How to Use

Panduan singkat untuk menjelajahi dashboard ini.

  1. Buka dashboard dan tinjau KPI utama di bagian atas.
  2. Gunakan filter (Kategori, Wilayah, Tahun) untuk mempersempit analisis.
  3. Baca visualisasi tren utama untuk memahami momentum penjualan atau keuntungan.
  4. Bandingkan penjualan dan keuntungan per kategori guna menentukan prioritas strategi.
  5. Periksa peta panas (*heatmap*) untuk mengidentifikasi wilayah atau produk dengan keuntungan tinggi.
  6. Verifikasi temuan menggunakan data tingkat lebih detail: sub-kategori, segmen pelanggan, dan kota.
